# Library-Management-System
A console-based application designed to manage the operations of a library.

## Features

### Librarian Features
- **Authentication**: Secure login for librarians.
- **Add Book**: Add new books to the library inventory.
- **Remove Book**: Remove books from the inventory.
- **View All Books**: Display the list of all books in the library.

### Member Features
- **Registration**: Register as a new member.
- **Login**: Authenticate existing members.
- **Borrow Book**: Borrow available books.
- **Return Book**: Return borrowed books.
- **View Borrowed Books**: See the list of books currently borrowed by the member.
- **View All Books**: Browse the entire library catalog.
